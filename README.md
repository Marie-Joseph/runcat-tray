# runcat-tray

Is a runcat port for Linux using libappindicator.

icons from: https://github.com/win0err/gnome-runcat

## requirements

- libappindicator-gtk3

## build

> mkdir build && cd build
> 
> cmake .. && make
> 
> make install

## start

> runcat
